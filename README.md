**Starhopper** is a browser-based retro arcade game where you pilot a spacecraft through the 12 Zodiac constellations to find extraterrestrial life.

Built using Google's **Antigravity** as a Proof of Concept (PoC) for an agentic coding course.

## ðŸŽ® How to Play

**Mission:** Connect all stars in each constellation sequence before time runs out. The time limit decreases with each level!

### Controls

| Key | Action |
| :--- | :--- |
| **Arrow Keys** | Move Ship |
| **Shift** | Boost Speed (2x) |
| **Z** | Brake / Slow Down (0.5x) |
| **Enter** | Start Game / Continue |

## ðŸš€ Deployment & Local Development

### Run Locally

Since this is a static site, you can simply open `public/index.html` in your browser.

Or, if you have the Firebase CLI installed:

```bash
firebase serve
```

### Deploy to Firebase

1.  Initialize Firebase (if not already done):
    ```bash
    firebase login
    firebase init
    ```
2.  Deploy:
    ```bash
    firebase deploy
    ```
